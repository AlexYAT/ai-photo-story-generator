# Demo example images

Набор тестовых входных изображений для демонстрации `photo_to_story` (GitHub, отчёты, ручные прогоны). Файлы лежат в `static/examples/`.

| Файл | Назначение |
|------|------------|
| `example_1_ordinary.png` | **Ordinary image** — обычная «фотографичная» сцена с понятным содержимым и композицией. Проверяет типичный сценарий: люди, интерьер, объекты. |
| `example_2_text_screenshot.png` | **Text screenshot** — имитация скриншота с текстом / интерфейсом (IDE, терминал). Проверяет описание интерфейсов и читаемого текста без выдумывания лишнего контекста. |
| `example_3_plain_background.png` | **Plain background** — много пустого фона и минимум объектов. Проверяет устойчивость к «пустым» кадрам и отсутствию богатой сцены. |
| `example_4_overloaded_collage.png` | **Overloaded collage** — плотная коллажная композиция, много мелких элементов. Проверяет разбор шумной сцены и приоритизацию деталей. |

## Запуск с примером (CLI)

Из корня репозитория (после настройки `.env`):

```bash
python main.py --image static/examples/example_1_ordinary.png --lang ru
```

Пути для остальных файлов аналогичны: замените имя файла на нужное.
